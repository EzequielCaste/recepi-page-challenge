## Overview

![screenshot](https://i.imgur.com/JSGpZgM.gif)

The demo for the solution to this challenge is hosted on GitHub Pages. I used CSS variables, Flexbox and Media Queries to make the page responsive for Mobile, Tablet and Desktop sizes. I start the page design mobile first andlater work on the desktop and tablet sizes.

## Features

<!-- List the features of your application or follow the template. Don't share the figma file here :) -->

This application/site was created as a submission to a [DevChallenges](https://devchallenges.io/challenges) challenge. The [challenge](https://devchallenges.io/challenges/TtUjDt19eIHxNQ4n5jps) was to build an application to complete the following user stories:

- [x] User story: I can see a recipe with ingredients and instructions
- [x] User story: I can select a checkbox if I have the ingredients
- [x] User story: I can see number of servings, baking times
